# Inversiones J&T - Sistema de Préstamos

Sistema moderno para la gestión de préstamos y operaciones financieras.

## Descripción

Este proyecto es una plataforma completa para administrar el flujo de efectivo, la contabilidad, el registro de clientes y gestionar los estados de cuentas de los préstamos. 

## Características Principales

- **Gestión de Cajas**: Flujo de dinero entre cajas chicas y la caja principal.
- **Reportes Financieros**: Reportes de flujo de efectivo, cobros atrasados y proyecciones.
- **Gestión de Clientes**: Registro y administración de clientes y garantes.
- **Autenticación y Roles**: Sistema de control de acceso.

## Stack Tecnológico

- **Framework**: Next.js 16 (App Router)
- **UI Components**: Shadcn UI & Tailwind CSS v4
- **ORM / Base de datos**: Drizzle ORM

## Cómo ejecutar el proyecto localmente

1. Clonar el repositorio
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Iniciar el entorno de desarrollo:
   ```bash
   npm run dev
   ```

El proyecto estará disponible en `http://localhost:3000`.
